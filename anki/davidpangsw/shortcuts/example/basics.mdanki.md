# Keyboard Shortcuts Basics

## Copy

What is the keyboard shortcut to copy selected text?

%

`⌘ + C` (Command + C)

Copies the selected text to the clipboard.

## Paste

What is the keyboard shortcut to paste from clipboard?

%

`⌘ + V` (Command + V)

Pastes the content from the clipboard.

[#keyboard](#) [#basic](#)

## Cut

What keyboard shortcut cuts the selected text?

%

`⌘ + X` (Command + X)

Removes the selected text and copies it to the clipboard.

[#keyboard](#) [#basic](#)

## Undo

How do you undo the last action?

%

`⌘ + Z` (Command + Z)

Reverts the most recent change.

[#keyboard](#) [#basic](#)

## Redo

What is the shortcut to redo an undone action?

%

`⌘ + Shift + Z` (Command + Shift + Z)

Reapplies the most recently undone action.
