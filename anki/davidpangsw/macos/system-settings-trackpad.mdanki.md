# macOS Trackpad Gestures

## Lab 1: Desktop Views Cycle

1. Show Desktop
2. Get back
3. Open Launchpad
4. Close Launchpad

%

1. Show Desktop (Spread with thumb and three fingers)
2. Get back (Pinch with thumb and three fingers)
3. Open Launchpad (Pinch with thumb and three fingers)
4. Close Launchpad (Spread with thumb and three fingers, or press Esc)

## Lab 2: Window and Space Management

1. Open Mission Control
2. Return
3. Show App Exposé
4. Return
5. Swipe between Desktops (or full-screen applications) - next
6. Swipe between Desktops (or full-screen applications) - previous

%

1. Open Mission Control (Swipe Up with Four Fingers)
2. Return (Swipe Down with Four Fingers, or click)
3. Show App Exposé (Swipe Down with Four Fingers)
4. Return (Swipe Up with Four Fingers, or click)
5. Swipe between Desktops (or full-screen applications) - next (Swipe Left with Four Fingers)
6. Swipe between Desktops (or full-screen applications) - previous (Swipe Right with Four Fingers)

## Lab 3: Notification Navigation

1. Open Notification Center
2. Close Notification Center

%

1. Open Notification Center (Swipe left from the right edge with two fingers)
2. Close Notification Center (Click outside or swipe right from edge)

## Lab 4: Page Navigation

1. Open a web browser or other apps with page-based navigation
2. Go back to previous page
3. Go forward to next page

%

1. Open a web browser or other apps with page-based navigation
2. Go back to previous page (Scroll Left with Two Fingers)
3. Go forward to next page (Scroll Right with Two Fingers)
